Flutter Dev Env setup on macOS

as of 2021-12-24

  • Install homebrew at https://brew.sh/

  • Install JetBrains Toolbox, VS Code and Flutter SDK with homebrew

  • Install Android Studio with JetBrains Toolbox installed above.

  • Install Xcode with App Store

  • Change Xcode selection with

    sudo xcode-select --switch /Applications/Xcode.app/Contents/Developer

  • Install Android SDK Command-line Tools with Android Studio

    In Preferences => Appearance & Behavior => System Settings => Android SDK => SDK Tools tab => Check Android SDK Command-line Tools (latest) and Apply

  • Agree to Android Licenses with

    flutter doctor --android-licenses
