棒球和我

今天又不停地问了女友很多关于棒球的问题，她也不厌其烦地给我讲。

距离第一次去甲子园看球也过去一个月了，我也觉得是时候写写我看棒球的事情了。

今天和女友说到，很奇怪，日语叫棒球叫野球，讲的是场地，中文叫棒球，讲的是击球用的道具，英文叫baseball，讲的是场上的三个方块儿，全都不一样。她说她好几年前就发现了。哈哈，不知道还有多少人发现了一样的事呢。

第一次去甲子园看球并不是我第一次去甲子园。在我陪女友第一次去关西考试顺便玩的时候，因为COVID导致场馆不能售票，我们只在它的博物馆里看了一个小时左右就走了。而真正第一次进甲子园看球，也是我第一次线下看棒球（虽然在那之前也没有认真看过线上比赛），是5月5日。阪神老虎主场打养乐多燕子。

那天是大晴天，如果没有女友给的棒球帽，太阳就会晒得我眼睛睁不开。因为买票买得很晚，我们只能坐在左边外野的山顶，虽然不能近距离观察球员，但是视野非常好。女友说她觉得外野的球员都很酷，可能也有这部分原因吧。

那个时候我一点都不懂棒球的规则，迷迷糊糊。（虽然在前一天我带她去买了手套和球作为生日礼物，也是一点都不懂什么样的手套才是好的手套，也迷迷糊糊。）不知道他们什么情况下会跑到哪里去，什么情况下会不跑了走回去。但是依然觉得现场气氛很热烈，大家都很开心。让我觉得来球场是一件很开心的事情。也许在对生活失去斗志的时候，来球场看一场比赛，这种气氛可以足够使人重新拾起信心吧。

那场比赛一开始燕子是领先的，但是在最后一局被老虎反超，3-2，获胜之后球场附近的商场里都在播放「六甲おろし」，比起5月31日我在明治神宫看的那场来说，气氛真的非常浓厚了（虽然歌听起来有几十年前的气氛，很像中国的红歌）。

甲子园的座位很小，也没有靠背，但是座位数量很多。神宫的座位有靠背，很舒服。（不知不觉开始比较了起来呢）

第二次看比赛是和同事[1]一起去的，本来是打算自己一个人去的，但是闲聊的时候聊到了，然后同时表示想一起去，然后刚好附近的座位空着（本来左右都想空着隔点人宽敞点），就一起来了。没想到当天有一个现场的同事[2]也来了（但是座位离很远）。

第二次看的比赛在5月31日。燕子和千叶罗德海洋。山田打出 Home run 之后再也没人得分，最后1-0结束，不怎么激烈的一场比赛。是不是因为两边的投手和捕手比较努力呢。但是现场看到つば九郎发呆。非常可爱。

最近买了6月25日和巨人的第二场的票。大概这次可以一个人看了。虽然有同事[3]当时也去，但是在内野。听说下个月同事 [2]有机会还要多组几个人去买团体票一起看。虽然应该人多点会很热闹，但是因为跟同事[1]看的比赛感觉比较尴尬，并不是很快乐。应该还是会习惯一个人看或者跟女友一起看。
